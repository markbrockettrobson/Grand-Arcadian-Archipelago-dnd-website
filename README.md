# The Arcadian Archipelago - Campaign Vault

An Obsidian vault for tracking the **Arcadian Archipelago** tabletop RPG campaign.

**Game System:** Pathfinder 2nd Edition (PF2e)

---

## What is this?

This repository serves as an interconnected wiki for managing and documenting an ongoing Pathfinder 2e home game. Using Obsidian's markdown format, all campaign elements (NPCs, locations, quests, session notes, etc.) are linked together for easy reference and navigation.

---

## Getting Started

### Using with Obsidian

1. **Install Obsidian** (if you haven't already)
   - Download from [obsidian.md](https://obsidian.md)
   - It's free for personal use

2. **Open this folder as a vault**
   - In Obsidian: "Open folder as vault"
   - Select this repository's root folder

3. **Start with Home.md**
   - This is your campaign dashboard
   - Contains links to all major sections
